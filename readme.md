# ROG STRIX Z790-A GAMING WIFI D4 Opencore
 华硕Z790吹雪D4版黑苹果opencore配置
 已测试Ventura与Sonoma
## 测试环境
CPU: Intel i7 13700KF  
显卡: 盈通6800XT花嫁  
内存: 64G 4x16阿斯嘉特女武神（） 
网卡:原装  
硬盘:爱国者P7000Z 
## 使用注意事项
* 请修改SMBIOS的序列号，参考三码生成教程
* Config中添加了屏蔽N卡独显的SSDT，如果你的第一槽为A卡请删除或替换
## 已知问题
* Sonoma下无法使用WiFi，需要等新版驱动
* CPU跑分比Windows偏低
